Installing the Mirabilis addon

Current De Rascal Repository Version: 2.2.0

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://De-Rascal.github.io/repo
  • Name it ".DeRascal" without the quotes
  • Head to "My Addons"
  • Select install an addon from zipfile.
  • When it asks for the location, select ".DeRascal", and install repository.de-rascal-2.2.0.zip.
  • Go back to My Addons and select "Install from repository"
  • Select the ".DeRascal" menu item
  • Open "Look-and-Feel" - "Skin" and install Mirabilis

Enjoy!
